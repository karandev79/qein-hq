<h3 align="center">QeinHQ - Studio Lamp</h3>

- A cool modular Studio Lamp for my room that has fully customizable lights and in-built speakers with reactive lights to music, sounds.

----

## Features

* Addressable RGB leds lights inside for 100% control over each LED.
* Double speakers with tweeter with bluetooth support.
* Modular desgin for easy storage and setting up.
* Reactive Lights to Music and Sounds.
